# Frontend - Next.js Static Site

This directory contains the Next.js frontend for the portfolio site.

## Development

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build  # Generates static site to out/
```

## Environment Variables

- `NEXT_PUBLIC_API_BASE_URL` - Backend API base URL (default: https://bxyzn.com/api)

See main README.md for more information.
